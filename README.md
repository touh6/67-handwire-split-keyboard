# 67
<img width="3546" height="2712" alt="full ds" src="https://github.com/user-attachments/assets/b2efd834-352d-49d8-989a-00887ed2a23c"/>
handwired split mechanical keyboard running on qmk firmware consisting of 12 keys 

## about the project:
this is handwired split mechanical keyboard made for the the love of the game (and tung tung tung sahur), it has a 3d printed enclosure that is shaped like the number 6 and 7. the heart of this device is the arduino pro micro, each embedded in each case of which there is a master and a slave controller. other components include cherry mx key switches, braille keycaps, diodes,m3 screws along with heat inserts.

## to build your own 67 keyboard:

buy all the components as mentioned in the [bill of materials](https://github.com/touh6/breadplayer/tree/main/bill%20of%20materials).

3d print all the required cases { files of which you can find in the [cad folder](https://github.com/touh6/breadplayer/tree/main/cad) }


and then, assemble it all in the case and you're good to go! :))
